## 💻 Projeto

O **Zingen** é uma página de um app de música no formato **One Page**, totalmente responsiva, ideal para diversos tipos de **micro, pequenas e médias empresas**.

### 🏗 Seções:

✅ Header  
✅ Hero  
✅ Conheça o app  
✅ Planos e preços  
✅ Download  
✅ Footer
